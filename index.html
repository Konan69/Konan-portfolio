About Me

I have successfully delivered high-quality software solutions utilized by thousands of individuals across the blue collar, fintech, lifestyle and transportation industries.

I specialize in backend engineering/development with proficiency in utilizing JavaScript/TypeScript (Node.js) with the Express.js framework and Python with Flask to create efficient, scalable server-side applications. I excel in seamlessly integrating backend systems with frontend applications.

On the frontend, I have enterprise level experience with React, HTML, JSX, Tailwind CSS, Material UI, and jQuery.

Additionally, I possess familiarity with Linux systems. In terms of database management, I've worked extensively with SQL and NoSQL databases such as MySQL and MongoDB. My expertise extends to supporting DevOps teams by implementing CI/CD using Jenkins & Fabric, containerization using Docker and Bash scripting for task automation. I also have skills in unit testing, Git for version control and utilization of Redis for temporary storage systems.
